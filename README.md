# Netflix Clone

A **Netflix clone** web application built with **React**, showcasing a responsive and interactive UI similar to Netflix. This project is for learning purposes and does **not contain real Netflix content**.

## Features

- Browse movies and TV shows by category
- Dynamic banners and sliders
- Search functionality
- Responsive design for mobile and desktop
- Movie trailers integration
- Dark/light mode (optional)

## Tech Stack

- **Frontend:** React, JavaScript, CSS
- **Styling:** Tailwind CSS / Bootstrap (choose what you used)
- **API:** TMDB (The Movie Database)
- **Other Libraries:** Axios, React Router, Movie Trailer, React YouTube
